# @pratik/agent-skills-dotnet

AI agent skill definitions for .NET/C# projects — part of the [`agent-skills`](https://github.com/pratik/agent-skills) monorepo.

## What's inside

| Skill | Description |
|-------|-------------|
| `core` | Non-negotiable principles for every task |
| `code-standards` | C# naming, DRY, no magic values |
| `design` | SOLID, KISS, YAGNI, pattern rules |
| `dotnet-best-practices` | .NET / EF Core async, cancellation, projection |
| `csharp-xunit` | xUnit testing guidelines |
| `mcp_dotnet` | MCP server tool rules |
| `dotnet-api` | RESTful API, OpenAPI, auth, versioning |
| `harness` | Validation scripts and CI hooks |
| `agents` | Architect + Developer agent personas |
| `memory` | Persistent lesson files for learning loops |

## Usage

This package is consumed by the CLI — do not import it directly.

```bash
npx @pratik/agent-skills dotnet-setup
```
